# Superpixel_Image_Classification_GCN_3834
A Graph Convolutional Network based model that uses images encoded via Superpixel graph networks to classify images. Made for a university term project.

# Dependancies/Libraries used
- Pytorch Geometric
- NetworkX
- Matplotlib
- Numpy
- Torchvision
